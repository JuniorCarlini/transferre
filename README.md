# YouTube Downloader

Script para baixar vídeos do YouTube via linha de comando.

## Requisitos

- **Python 3** - [Download](https://www.python.org/downloads/)
- **yt-dlp** - `pip install yt-dlp` ou `brew install yt-dlp`
- **Node.js** (runtime JS necessário pelo yt-dlp) - `brew install node`
- **ffmpeg** (para mesclar vídeo+áudio) - `brew install ffmpeg`

## Uso

```bash
python3 baixar_youtube.py
```

Cole a URL do vídeo e escolha a qualidade desejada.
